하네스의 4축

"모델 밖으로 무엇을 꺼낼까"의 답을 4가지 카테고리로 분류한다.
좋은 회사 운영에 비유하면 — 업무 노트 · 매뉴얼 · 보고 체계 · 결재선이다.
📦
AXIS 1
Memory
안정적으로 보관해야 할 상태. Working / Semantic / Episodic / Personalized.
회사 비유: 업무 노트, 회의록, 기록물
📋
AXIS 2
Skills
반복해서 쓰는 절차·판단·규칙. 매번 즉석에서 만들지 않도록 패키징.
회사 비유: 표준 매뉴얼, 체크리스트
🔗
AXIS 3
Protocols
상호작용의 계약. Agent-User · Agent-Agent · Agent-Tools 3가지 표면.
회사 비유: 보고 체계, 협업·인수인계 규칙
🛡️
AXIS 4
Mediating Mechanisms
흐름을 통제하고 되돌리는 장치. Sandbox · Observability · Compression · Eval · Approval · Sub-agent.
회사 비유: 결재선, 감사, 품질 검사
새 기능을 어디에 넣을지 묻는 질문: "기억? 절차? 상호작용? 통제?" — 이 4축으로 답이 자동으로 나온다.
