Stage 4·5 — + Subagent + Permission

자기평가 편향을 깨는 reviewer subagent + 위험 행동을 시스템이 차단하는 권한 정책. 마지막 두 계층.
4
STAGE 4
+ Subagent
"같은 세션의 모델은 자기 가정에 끌린다." 새 세션의 reviewer가 객관적으로 본다.
RESEARCHER
읽기 전용 · Read · Grep · Glob
BUILDER
파일 수정 · Read · Write · Edit · Bash
REVIEWER
새 세션 · 비판적 검토 · 보고만
5
STAGE 5
+ Permission
"하지 마세요"는 약하다. "못 하게 만든다"가 강하다. Allowlist > Denylist.
ALLOW
Read · Grep · Glob
ASK
Edit · Write · Bash
DENY
Edit **/.env*
DENY
Bash *DROP TABLE*
DENY
Bash rm -rf*
5단계 완료. 프롬프트는 한 줄 그대로, 결과는 프로덕션 수준. 누가·언제·어떤 세션이어도 같은 품질.
