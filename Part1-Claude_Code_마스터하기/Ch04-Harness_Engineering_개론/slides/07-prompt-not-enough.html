왜 프롬프트만으론 부족한가

"조심해서 운전하세요"라는 말만으로 안전한 운전이 되지 않는다. 차선·신호등·브레이크·안전벨트가 필요하다.
🗣️
프롬프트 (운전자에게 하는 말)
"보안 지켜"
→
비밀 유출이 막히지 않는다
"테스트해"
→
실제 테스트가 실행되지 않는다
"조심해"
→
위험 명령을 그대로 실행할 수 있다
"기억해"
→
세션이 끝나면 사라진다
vs
🛤️
하네스 (도로 · 신호 · 차량)
차선·신호등
↔
CLAUDE.md · 컨벤션 문서
브레이크·벨트
↔
Permissions · Hooks
운전면허
↔
평가 하네스 (Eval)
보험·정비
↔
거버넌스 · Audit log
프롬프트는 무가치한 게 아니다. 그것 하나로는 충분하지 않다. 하네스 = "사람에게 부탁하는 것 → 시스템이 보장하는 것"으로의 전환.
