Harness의 뿌리

"복잡한 요소를 흩어지지 않게 묶고, 필요한 방향으로 힘이 흐르게 만드는 구조"

🚗
자동차 배선 하네스
수천 가닥 전선을 묶어 정리, 과전류·마모 방지, 정비 추적 가능
🧗
등산 안전벨트
움직임은 그대로, 추락 시에만 정확히 받쳐주는 장비
🐎
말의 마구
힘을 줄이는 게 아니라, 원하는 방향으로 쓰게 해주는 도구
🧪
Test Harness (ISTQB)
실제를 그대로 쓸 수 없을 때, 통제된 환경으로 검증하기
