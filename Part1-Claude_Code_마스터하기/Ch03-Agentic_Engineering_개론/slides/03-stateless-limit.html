Vibe Coding Mechanics
🗂️Stateless
모든 prompt가 독립적. 어제 결정한 것, 지난 PR의 합의, 팀 컨벤션 — 어느 것도 다음 prompt에 자동으로 들어가지 않음. 매번 처음부터.
📋Manual Copy-Pasting
인간이 probabilistic output(확률적 결과물)을 손으로 옮김. ChatGPT → IDE 코드 복사 → 오류 메시지 → 다시 ChatGPT. 이해도 못한 채 복붙.
