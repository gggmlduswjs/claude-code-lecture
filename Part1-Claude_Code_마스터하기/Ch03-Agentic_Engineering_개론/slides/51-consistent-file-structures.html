Principle 2 · Structural Consistency

Consistent File Structures — Predictively Locate

디렉토리 hierarchy를 표준화 — agent가 "어디에 두지?"를 묻지 않고 예측적으로 정확한 자리에 둔다
"Standardizing directory hierarchies so the agent can predictively locate and generate files."
디렉토리 hierarchy를 표준화 — agent가 파일을 예측적으로 찾고 생성할 수 있도록.
Top-Level · Domain Modules
src/
├── payment/ ← 결제는 항상 여기
├── auth/ ← 인증은 항상 여기
├── user/ ← 사용자는 항상 여기
├── billing/ ← 청구 도메인
└── notification/ ← 알림 도메인
Per-Module · Identical Layout
src/payment/
├── index.ts ← public API
├── handlers/ ← request handler
├── services/ ← business logic
├── repositories/ ← data access
└── __tests__/ ← test
