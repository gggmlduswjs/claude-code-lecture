LogQL Query Example
결제 완료 이벤트 중 — 금액 100만원 초과만 추출
# Grafana Loki — LogQL
{app="payments"}
  |= "payment_completed"
  | json
  | amount > 1000000
  | line_format
      "{{.user_id}} → ${{.amount}}"
agent가 자기 작업 후 이 쿼리를 돌려서 — "내가 만든 결제 코드가 비정상적으로 큰 금액을 만들지 않았나?" 직접 확인.
Self-Correction Loop
5-단계 검증 사이클 — 2-layer 통합
1
agent가 작업 완료 — 코드 작성·변경
2
Layer 1 — unit / integration test 통과 (pass/fail 검증)
3
Layer 2 — LogQL 쿼리로 내부 state 검증
4
어느 하나라도 실패 → self-correction loop 진입
5
둘 다 통과해야 "validated work"로 인정 — 비로소 production-grade
