자산 1
Shared Skills
.claude/commands/
팀이 commit하는 슬래시 명령 모음. 새 엔지니어가 입사하면 — 첫날부터 이 명령들을 쓴다. 온보딩 시간이 극적으로 줄어든다.
/pr-review /deploy-staging /db-migrate /incident-triage
🔌
자산 2
Shared MCP Servers
team-internal MCPs
팀 내부 시스템을 표준화된 MCP 서버로 노출. 한 번 만들면 모든 엔지니어가 동일 인터페이스로 접근.
mcp-internal-wiki mcp-incident-history mcp-vendor-portal
📋
자산 3
Shared CLAUDE.md
team-wide templates
각 프로젝트의 CLAUDE.md를 공통 템플릿에서 시작. 결제 · 인증 · frontend 등 표준 골격을 팀 메타 repo에 둔다. 신규 프로젝트는 fork로 시작.
payment.md auth.md frontend.md
🛡️
자산 4
Shared Golden Rules
GOLDEN_RULES.md (team-wide)
위반 금지 룰의 마스터 리스트. "이건 절대 안 됨"의 팀 차원 합의. Pillar IV의 GOLDEN_RULES를 team-wide standard로 격상.
no-direct-db-write no-secret-in-log no-skip-migration
🔗 각자 자기 영역만 책임지는 게 아니라 — 상호 참조한다. Skill이 MCP를 호출, MCP가 Golden Rules를 검사, Rules가 CLAUDE.md 템플릿을 가리킨다. 진정한 graph가 형성
