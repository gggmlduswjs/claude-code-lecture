Skill Automation

Skill Automation — 4가지 패턴

수동 워크플로우(예: 포털 클릭)를 agent가 접근 가능한 Skill / API endpoint로 변환
🖱️
Pattern 01
포털 → CLI
AWS 콘솔 클릭→aws iam CLI
IAM 권한 조정 — aws iam CLI를 묶은 /grant-readonly Skill.
🚀
Pattern 02
Deploy → Script
Stripe dashboard→stripe CLI
환불 처리 — stripe refunds create를 묶은 /issue-refund Skill.
🔍
Pattern 03
Lookup → MCP
Notion 페이지→Notion API MCP
회의록 작성 — Notion API를 묶은 /post-meeting-notes Skill.
⚖️
Pattern 04
승인 → Policy
"안전한가요?"→policy.md
사람 결정 대기를 명시적 policy로 codify — agent가 자체 판단.
CLI는 단발 도구. Skill로 codify하면 — 다른 사람도 즉시 쓸 수 있는 팀 자산. 그리고 agent에게 직접 노출되어야 함.
