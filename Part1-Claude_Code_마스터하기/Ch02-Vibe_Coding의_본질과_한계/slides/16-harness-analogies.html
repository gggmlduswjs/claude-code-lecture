다른 분야의 교훈

강력한 힘을 제어하는 시스템은 이미 다른 영역에 답이 있다
🐎
Lesson 1
마구 (Harness)
문제
말은 강력한 동물이지만, 풀어놓으면 원하는 방향으로 가지 않는다.
해법
마구를 씌우면 말의 힘을 제어할 수 있다. 능력을 줄이는 게 아니라 원하는 방향으로 발휘하게 만드는 것.
Claude Code → 마구가 없으면 강력하지만 통제되지 않는다
🔄
Lesson 2
CI/CD 파이프라인
문제
개발자가 코드를 직접 서버에 올리던 시절 — 사람의 실수가 곧 장애였다.
해법
커밋하면 자동으로 테스트·빌드·배포가 실행되며 품질을 보장하는 시스템.
AI 코딩 → 같은 패턴 — 출력을 시스템으로 검증하는 파이프라인
두 분야 모두 같은 답에 도달했다 — "제어와 구조". AI 코딩의 답도 같다.
