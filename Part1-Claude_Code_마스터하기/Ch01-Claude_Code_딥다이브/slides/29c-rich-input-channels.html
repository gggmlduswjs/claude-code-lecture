풍부한 입력 채널

텍스트 외 4가지 통로 — 컨텍스트를 직접 주입한다
@
파일 참조
파일명 앞에 @를 붙이면 Claude가 즉시 그 파일을 읽고 컨텍스트에 포함한다.
> @src/auth/session.ts 와
  @src/middleware/auth.ts 비교,
  인증 흐름 정리.
🖼
이미지 드래그
스크린샷·디자인 시안을 입력창에 드래그/붙여넣기. UI 작업·디버깅에 가장 효과적인 입력.
> [스크린샷 첨부]
  이 모달의 정렬을
  시안과 일치하게 수정해줘.
🔗
URL 참조
웹 페이지·문서를 직접 페치. 자주 쓰는 도메인은 /permissions로 화이트리스트.
> https://docs.anthropic.com/...
  에서 prompt caching
  예시 코드를 가져와줘.
|
데이터 파이프
CLI에서 표준 입력으로 데이터를 흘려보낸다. 로그·테스트 출력 분석에 적합.
$ cat error.log | claude
  $ kubectl logs ... |
  claude -p "패턴 요약"
💡 텍스트로 설명하는 대신 원본을 주입하라 — 컨텍스트 손실 없이 정확한 추론.
