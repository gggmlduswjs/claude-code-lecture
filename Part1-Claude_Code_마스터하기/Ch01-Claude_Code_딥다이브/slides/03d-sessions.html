세션 — 모든 작업의 단위

메시지 · 도구 호출 · 결과가 JSONL로 남는다
~/.claude/projects/<프로젝트id>/sessions/*.jsonl
→ 그래서 되감기 · 잇기 · 분기가 가능
/rewind
체크포인트 복원
이전 시점으로 되감기 — 파일 변경까지 함께 복원. 잘못된 방향 빠르게 폐기.
claude --continue
최근 세션 잇기
가장 최근 세션을 그대로 이어 작업. 한 줄로 다시 진입.
claude --resume
세션 목록에서 선택
과거 세션 목록을 띄우고 골라서 재진입. 어제·지난주 세션도 OK.
/branch
같은 시점 분기
--fork-session과 동일. 같은 컨텍스트에서 다른 방향을 동시에 시도.
각 세션은 독립 — 새 세션은 깨끗한 컨텍스트에서 시작. 자동 메모리(MEMORY.md) + CLAUDE.md가 세션 간 지속성을 책임진다.
