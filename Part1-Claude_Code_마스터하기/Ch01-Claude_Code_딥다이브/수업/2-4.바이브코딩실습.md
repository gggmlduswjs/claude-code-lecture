

은행 거래내역 CSV를 업로드하면 표로 보여주는 분석 앱을 만들어줘.
- 컬럼: 날짜, 설명, 카테고리, 금액
- 업로드는 드래그앤드롭 + 클릭 둘 다 지원
- CSV 파싱은 papaparse 사용 (필요하면 설치)
- 표는 Tailwind로 깔끔하게, sticky header
- App.tsx 한 파일에서 끝내줘

결과: papaparse 자동 설치 + App.tsx 작성 → 드롭하자 200줄 표가 1초 만에 렌더.

표 위에 차트 2개 추가해줘.
1. 카테고리별 지출 비중 — 도넛 차트
2. 월별 지출 추이 — 막대 차트
recharts 라이브러리 써서, 둘이 가로로 나란히.
색상은 부드러운 톤으로.

결과: recharts 설치 + 데이터 집계(groupBy + sum) + 두 컴포넌트 + 반응형 레이아웃 자동 처리.

페이지 상단에 큰 숫자 카드 3개 추가해줘.
- 이번 달 총 지출
- 가장 많이 쓴 카테고리 (이름 + 금액)
- 평균 일일 지출
숫자는 ₩ 통화 포맷, 천 단위 콤마.
"이번 달"은 CSV에서 가장 최근 월 기준.

결과: "₩2,847,300 / 식비 ₩980,100 / 일평균 ₩94,910" 같은 인사이트가 페이지 열자마자 노출.
