사진 공유 SNS 앱을 만들 거야. 핵심 기능만 정리하면:
1. 사용자 회원가입/로그인 (이메일 기반)
2. 사진 업로드 (최대 5MB, JPEG/PNG)
3. 피드 — 최신순 사진 목록
4. 좋아요 기능

기술 스택: Next.js 15, Supabase (Auth + DB + Storage), Tailwind CSS
먼저 DB 스키마를 설계하고 보여줘. 구현은 아직 하지 마.

1. 시작 전에 스펙 쓰기	2~3분, 프로젝트·기능·스택·제약을 한 블록에 정리	일관된 방향성
2. 한 번에 하나씩	스키마 → API → UI → 통합 테스트 순으로 분리 진행	각 단계 검증 가능
3. 결과 확인 후 피드백	실제 브라우저로 보고 다음 프롬프트 작성	컨텍스트 오염 차단
