# Agentic Engineering
"로그인 기능을 구현하려고 해.
/plan 모드로 전환해서, 먼저 구현 계획을 보여줘.
계획에는 다음이 포함되어야 해:
 1. 필요한 DB 테이블
 2. API 엔드포인트 목록
 3. 프론트엔드 컴포넌트
 4. 보안 고려사항
각 단계별로 내 승인을 받은 후에 실행해."

Harness:      프로젝트 규칙으로 지시 (영구적)

# CLAUDE.md

## 코딩 규칙
- TypeScript strict 모드 사용
- 모든 함수에 return 타입 명시
- any 타입 사용 금지
- 에러 처리는 Result 패턴 사용

## 테스트
- 모든 API 엔드포인트에 테스트 필수
- vitest 사용 (jest 사용 금지)

## 보안
- 환경변수는 반드시 .env 사용
- SQL은 항상 parameterized query 사용
- API에 반드시 인증 미들웨어 적용
