Habit 01 of 05

짧고 집중된 세션

가장 흔한 실수는 한 세션에서 너무 많은 것을 하려는 것. 긴 대화는 컨텍스트 로트(context rot)를 만듭니다.
BAD PATTERN
한 세션에 모든 걸
✗한 세션에서 5~6개 작업을 연달아
✗대화가 50번 왕복 → 자동 컴팩션 발생
✗초반 지시사항이 요약에서 사라짐
✗"try-catch로 해줘" 했는데 결과에 빠짐
→
GOOD PATTERN
하나의 세션 = 하나의 작업
✓명확한 목표를 정하고 그것만
✓새 작업은 새 세션에서 시작
✓/compact를 선제적으로 사용
✓초반 지시사항이 세션 끝까지 살아있음
추천: 세션 컨텍스트가 30~40%를 넘기지 않도록 관리. 자동 컴팩션 기다리지 말고 /compact로 선제 압축.
