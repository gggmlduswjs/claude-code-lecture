CLAUDE.md vs Memory

Auto-Memory는 대화 중 학습한 패턴을 ~/.claude/projects/ 하위 markdown 파일로 저장. 다음 세션에서도 유지됩니다.
📄
CLAUDE.md
TEAM LEVEL · GIT COMMIT
▸프로젝트 수준의 규칙
▸팀 전체가 공유하는 컨벤션
▸아키텍처 · 빌드 명령어 · 네이밍
▸"any 타입 금지" 같은 절대 규칙
VS
🧠
Memory
PERSONAL · AUTO-SAVED
▸개인별 선호도와 학습 내용
▸자연어 요청으로 저장 ("기억해: ...")
▸Claude가 중요하다 판단하면 저장
▸Git에 커밋하지 않는 개인 자산
"기억해: 이 프로젝트에서 API 응답은 항상 { data, error, meta } 형태를 따라"
