Hooks — 팀 자동화

Claude Code의 특정 동작 전후에 자동 실행되는 셸 스크립트. .claude/settings.json에 정의하고 Git에 커밋하면 팀 전체 적용.
✨
자동 포맷팅
Claude가 파일을 수정하면 자동으로 Prettier 실행. 팀원이 기억할 필요 없이 시스템이 보장.
Hook
PostToolUse → prettier --write
🛑
위험 명령 차단
rm -rf, git push --force 같은 위험한 명령어를 사전 차단.
Hook
PreToolUse → check && block
🔍
린트 검증
코드 변경 후 ESLint 자동 실행. 컨벤션 위반을 그 자리에서 잡아냅니다.
Hook
PostToolUse → eslint --fix
